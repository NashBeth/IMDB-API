# IMDB-API

A python script which takes imdb ID as input and outputs a JSON object of all the information available on imdb.com

# Dependancies
* BeautifulSoup
* urllib2
* json

# Usage

<b>Input:</b><br>
Enter IMDB ID: tt23954271
